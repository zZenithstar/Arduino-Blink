# Arduino-Blink

This is a simple Arduino-LED work.First, all the LEDs light up, and then they light up in sequence;
the functionality was implemented using clean, well-structured code.


---ITEMS---
-Arduino UNO
-BreadBoard
-5 Jumper Cable(M-M)
-4 LED
-4 Resistor(120ohm %1)

---Connections--- 
-LED1(7) 
-LED2(6) 
-LED3(5) 
-LED4(8)
